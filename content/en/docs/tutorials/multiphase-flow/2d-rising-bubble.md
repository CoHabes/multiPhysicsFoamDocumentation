## Overview

A gas bubble rises through a quiescent liquid under buoyancy. This 2D
axisymmetric case is a standard benchmark for sharp-interface multiphase flow
solvers and validates surface-tension-driven interface dynamics.

## Running the case

```bash
cd $FOAM_RUN/../multiPhysicsFoam/tutorials/multiphaseFlow/2dRisingBubble
./Allrun
```
