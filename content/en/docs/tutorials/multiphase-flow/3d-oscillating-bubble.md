## Overview

A 3D bubble subject to surface-tension restoring forces undergoes shape
oscillations. This case tests the ability of the capillary interface model
to capture dynamic deformation of a free interface in three dimensions.

## Running the case

```bash
cd $FOAM_RUN/../multiPhysicsFoam/tutorials/multiphaseFlow/3dOscillatingBubble
./Allrun
```
