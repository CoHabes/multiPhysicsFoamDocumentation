## Overview

A 2D stirred vessel with a rotating impeller region connected to a stationary
outer region through an arbitrary mesh interface (AMI). This case demonstrates
rotating machinery support within the `pimpleFluid` region type.

## Running the case

```bash
cd $FOAM_RUN/../multiPhysicsFoam/tutorials/singleRegion/pimpleFluid/mixerVesselAMI2D
./Allrun
```
