These cases use the `pimpleFluid` region type in single-region mode, equivalent
to running a standard OpenFOAM `pimpleFoam` simulation through the
`multiPhysicsFoam` infrastructure.

## Cases

- [Pitz Daily](pitz-daily/) — turbulent flow over a backward-facing step
- [Mixer Vessel AMI2D](mixer-vessel-ami2d/) — rotating mixer with an arbitrary mesh interface
